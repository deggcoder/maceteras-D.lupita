Selección y molida

El proceso de elaboración comienza cuando se va a la mina a seleccionar y recoger el barro en piedra. Una vez seleccionado, se deben quitar todas las hierbas y piedras para que se pueda comenzar la elaboración.

Una vez el barro esté completamente limpio, se pasa por los molinos añadiéndole agua en función de las macetas que se quieran realizar, dado que cuanto más altas sean las macetas que se van producir más duro debe estar el barro, sino se corre el riesgo de que no se pueda trabajar.

Amasado

A continuación, cuando se tiene la arcilla totalmente pasada por los molinos, se procede a la siguiente fase que es el amasado. Esta es una fase importantísima en el proceso, el amasado debe ser uniforme ya que si el barro no está bien amasado y lleva bolsas de aire se corre el riesgo de que las piezas estallen en el horno.

Producción y secado

Después, se coloca la peña de barro en el torno y se pasa a la realización de las piezas de forma artesanal, es decir, el alfarero va modelando las macetas con las manos.

Conforme se van realizando las macetas, se van poniendo en una tabla o solera para que sequen de forma gradual, ya que si el secado es muy rápido puede llevar a que las macetas se agrieten, dado que pierden muy rápido la humedad.

Lijado y horneado

Una vez secas, las macetas se lijan y se meten en el horno. Esta cocción está entre las ocho y diez horas de duracion y, una temperatura aproximada de 1.000 ºC. Cuando se termina de cocer la hornada se deben dejar enfriar las macetas de forma lenta, porque si se apresura el enfriamiento, se corre el riesgo de que las macetas se agrieten y se rompan.

04
05
Decorado y 2do horneado

Las macetas se bañan en un recipiente lleno de esmalte del color seleccionado. Una vez bañadas se raspa la base para que no se peguen en el horno y se quitan las posibles rebabas o poros que pudieran tener para poder pintarlas sin problemas.

Luego, se coloca la maceta en una mesa y se decora con los distintos colores seleccionados. Una vez pintadas se colocan en soleras para la 2da cocción, que será a 950 ºC.

Con esto, ya está totalmente terminada la maceta para su comercialización.
